Sticky columns

If you have a wide table that is scrollable and want to make a column stick, use the sticky table column options. You can use the sticky option to stick the column to the left of the table and stickyEnd to stick the column to the right of the table. The column will stick once it passed the left (sticky) or right (stickyEnd) border of the table. To make the button column that comes with the form functionalities sticky, use the simplemattable stickyButtons option.

Typescript - Data

this.dataSimple = [
  new ComplexTestData(1000000, 400000000, 'test1', null, 'test2'),
  new ComplexTestData(2, 42, 'test2', null, 'test3'),

  new ComplexTestData(4, 44, 'test2', null, 'test3'),
