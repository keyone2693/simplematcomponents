Internationalization

Simplemattable uses Angular Material DataTable, so the internationalization classes of the corresponding features can be used:
  • MatSortHeaderIntl for sorting button Label
  • MatPaginatorIntl for paginator button texts

You can provide your own implementation of the classes above as shown in this Stackoverflow thread.

For internationalization of the date input form field, see the Angular Material Docs

Simplemattable itself only has two labels which you would have to provide for internationalization:
  • Filter placeholder/label on simplemattable. Change this via the filterLabel input parameter. Default is 'Filter'.
  • Column filter placeholder/label on table column. Change this via the table column option colFilterLabel. Default is 'Filter'.
