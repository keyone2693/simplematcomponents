Column Visiblility and Responsive Design

You can hide whole columns or just the text of a column on small (sm) or very small (xs) screens.
Hiding only the text can be useful if you only want to display an icon with no text on smaller screens.
Additionally, you can manually toggle the visiblility of a column by modifying its visibility property.

Typescript - Data

this.dataVisibility = [
  new ComplexTestData(1, 40, 'test1', new TestData('Key1', 'Value1', d1), 'test2'),
  new ComplexTestData(2, 42, 'test2', new TestData('Key2', 'Value2', d2), 'test3')
];

Typescript - Columns

this.columnsVisibility = [
  new TableColumn<ComplexTestData, 'id'>('My ID - Hidden Text on xs/sm', 'id')
    .isTextHiddenSm(true),
  new TableColumn<ComplexTestData, 'value'>('My Value - Hidden Column on xs/sm', 'value')
    .isHiddenSm(true),
  new TableColumn<ComplexTestData, 'description'>('My Description - Manual Toggle', 'description'),
];

HTML

<button mat-raised-button (click)="columnsVisibility[2].visible = !columnsVisibility[2].visible">
Toggle Description-Column
</button>
<smc-simplemattable [data]="dataVisibility" [columns]="columnsVisibility"></smc-simplemattable>
